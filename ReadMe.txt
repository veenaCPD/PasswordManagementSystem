
Installation

1. Install sqlite

2. Install mysql connector
python -m pip install mysql-connector-python

3. Install tkinter
pip install tk

4. Install pyperclip for copy and paste clipboard functions
pip install pyperclip

5. Install pillow as Tkinter relies on Pillow for working with images.
pip install pillow
